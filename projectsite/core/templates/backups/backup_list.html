About Backups
  • Automated Backups: The system creates automatic backups. You can also create manual backups.
  • Restore Point: Before restoring a backup, the current database is automatically saved as a restore point.
  • Cleanup: Remove old backups to save disk space. Backups older than 30 days are typically removed.
  • Compressed Storage: Backups are compressed with gzip for efficient storage.
  • Recovery: In case of data loss, you can restore to any point in time using available backups.
